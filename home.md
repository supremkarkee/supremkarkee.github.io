##  Project Overview
This technical journal documents the configuration, security hardening, and performance analysis of a headless Linux server.

**Core Objectives:**

* **Architecture:** Deploying a dual-system environment (Server + Workstation) via VirtualBox.
* **Security:** Implementing SSH hardening, Firewalls (UFW), Fail2ban, and Mandatory Access Control.
* **Performance:** Analyzing CPU/RAM usage under load and optimizing resource efficiency.
